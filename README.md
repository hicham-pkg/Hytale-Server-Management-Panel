# Hytale Server Management Panel

A self-hosted web panel for managing a Hytale server on Ubuntu.

## Quick Start

```bash
git clone https://github.com/hicham-pkg/Hytale-Server-Management-Panel.git
cd Hytale-Server-Management-Panel
sudo ./install.sh
```

## Repo Layout

- `hytale-panel/` — production monorepo (API, web, helper, shared package, docs, deploy scripts)
