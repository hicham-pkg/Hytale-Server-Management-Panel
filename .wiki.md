# Project Summary
The Hytale Server Management Panel is a comprehensive web application designed to streamline the management of Hytale game servers, primarily on Ubuntu operating systems. It offers an intuitive interface suitable for both novices and experienced users, facilitating critical server administration tasks such as monitoring server health, managing player access, and executing operations like backups and bans. Leveraging modern web technologies, this panel enhances the server management experience, allowing users to effectively oversee their gaming environments.

# Project Module Description
The project consists of several functional modules, each tailored for specific tasks:
- **Authentication**: Manages user login, registration, and two-factor authentication (2FA).
- **Server Management**: Controls server operations (start, stop, restart) and monitors performance metrics.
- **Player Management**: Administers player access through whitelists and bans.
- **Backup Management**: Handles the creation, restoration, and management of server data backups.
- **Crash Reporting**: Logs crash events for further analysis.
- **Audit Logging**: Records administrative actions for security and compliance.
- **Settings Management**: Offers configuration options for the panel and server, preventing dangerous settings.

# File Description Inventory
- **deploy/**: Scripts for deploying and managing the panel.
- **docs/**: Documentation files for installation, operations, security, and upgrades.
- **nginx/**: Configuration files for Nginx reverse proxy.
- **systemd/**: Service files for managing game server and helper service.
- **.env.example**: Template for environment variables.
- **docker-compose.yml**: Configuration for Docker services.
- **install.sh**: Script to set up the server and panel environment.
- **packages/**: Contains the main codebase, including API and helper services.

# Technology Stack
- **Frontend**: Vite, React, React Router, Tailwind CSS
- **Backend**: Fastify, TypeScript
- **Database**: PostgreSQL
- **Deployment**: Docker, Nginx
- **Authentication**: JWT, TOTP for 2FA

# Usage
To set up the Hytale Server Management Panel, follow these steps:
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd hytale-panel
   ```
2. Run the installation script:
   ```bash
   sudo ./install.sh
   ```
3. Configure the environment:
   Edit the `.env` file to set your domain and secrets.
4. Start the services:
   ```bash
   docker-compose up -d
   ```
5. Run database migrations:
   ```bash
   docker-compose exec api node dist/db/migrate.js
   ```
6. Create the admin user using the seed script:
   ```bash
   DATABASE_URL=postgresql://hytale_panel:<DB_PASSWORD>@localhost:5432/hytale_panel npx tsx packages/scripts/seed.ts
   ```
